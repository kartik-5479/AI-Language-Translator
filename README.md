# LinguaLift AI Translator

A polished browser-based language translator website with phrase shortcuts, copy and speech controls, local translation history, and graceful demo fallback when the live translation request is unavailable.

## Features

- Translator workspace with language swapping, copy, speech, and quick phrases.
- Dark mode toggle that remembers your preference.
- Separate `history.html` page for saved translations, with CSV download.

## Run

Recommended:

```bash
python app.py
```

This starts the website at:

`http://127.0.0.1:8000/`

No Flask install is required.

On Windows, you can also double-click `RUN_WEBSITE.bat`.

You can also open this file directly:

`translator_project/index.html`
